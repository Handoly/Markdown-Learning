# Markdown 学习

## 一、说明

本仓库用来存放学习`Markdown`的一些操作方法与技巧。

## 二、常用快捷键

- `ctrl+0` ：段落  

- `ctrl+1` ：一级标题
- `ctrl+2` ：二级标题
- `ctrl+3` ：三级标题
- `ctrl+4` ：四级标题
- `ctrl+5` ：五级标题

- `ctrl+Shift`+`:  转换为代码
- `ctrl +Shift+k`: 插入代码块
- `ctrl+[`:列表等级提高

- `ctrl+]`:列表等级下降

- `ctrl+Shift+[`：有序列表

- `ctrl+Shift+[`：无序列表

- `ctrl+b`:文本加粗

- `ctrl+Shift+i`：插入图片
